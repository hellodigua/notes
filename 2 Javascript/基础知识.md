# 基础知识

## 数组

### 类型检测

typeof [] // object

[] instanceof Array // false

Array.isArray([]) // true

## 常用技巧

### 数组合并

- concat 返回合并后的新书组

var c = a.concat(b)

- push 保存到原数组内

Array.prototype.push.apply(a,b)

### 数组深拷贝

把数组赋值给一个变量，只是复制了一个指向数组的指针（又叫引用）
当我们改变原数组的值，新数组的值也跟着改变
为了避免这样，你需要对数组进行深拷贝

- concat/slice （针对一维数组）

- 暴力深拷贝

for (var x = 0; x < arr.length; x++){
    newArr.push(arr[x])  // 用循环逐个把值保存在新数组内
}
