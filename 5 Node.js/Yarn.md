# yarn

## 安装

npm install yarn -g

## 常用命令

- 初始化项目
  yarn init(npm init)

- 添加包
  yarn add [package] --dev （开发依赖）
  npm install [package]

- 更新包
  yarn upgrade [package]
  npm update [package]

- 安装项目里的包依赖

yarn install

- 移除本地依赖包

yarn remove [package]
npm uninstall [package]
